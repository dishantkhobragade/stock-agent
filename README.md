# 📈 Stock Analysis AI Agent

An AI-powered stock analysis tool built with Python, Gemini AI and Streamlit.

## 🤖 What It Does
- Search any Indian or global stock by name
- View 5-year interactive price history chart
- Get key metrics (PE, PEG, ROE, Debt/Equity, EPS)
- Receive AI-powered analysis and 6-month prediction powered by Gemini AI

## 🛠️ Built With
- Python
- Gemini AI (google-genai)
- yfinance (stock data)
- Streamlit (web UI)
- Plotly (interactive charts)

## 🚀 How To Run
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Add your Gemini API key in config.py
5. Run the app
